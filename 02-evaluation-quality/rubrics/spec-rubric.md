Skill 2 — Evaluation Rubric: Project Specs

Here's a rubric you can reuse on any project:

| Dimension | What to look for | Score (1–5) |
|---|---|---|
| Clarity of goal | Is the end state unambiguous? | |
| Audience definition | Do you know exactly who this is for? | |
| Scope boundaries | Clear must-have AND must-not-have? | |
| Success criteria | Measurable, not vague? | |
| Constraints | Tech, budget, time limits stated? | |
| Tone & voice | Is the desired output voice defined? | |
| Actionability | Could an agent start work immediately? | |
